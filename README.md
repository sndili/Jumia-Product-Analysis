# Jumia-Product-Analysis

# Project Overview

This project analyzes product performance on Jumia using Microsoft Excel. The goal was to understand how pricing, discounts, customer ratings, and reviews influence product performance and to present the findings in an interactive dashboard.

# Dataset
The dataset contains information on Jumia products, including:
- Product Name
- Current Price
- Old Price
- Discount Percentage
- Number of Reviews
- Product Rating
- Product Category

# Data Cleaning
The following data preparation steps were performed:
- Removed duplicate records
- Checked for missing values
- Converted price columns to numeric format
- Standardized rating values
- Calculated discount amounts
- Created rating categories (Poor, Average, Excellent)
- Created discount categories (Low, Medium, High)

# Analysis Performed
The analysis focused on:
- Average product prices, ratings, and discounts
- Most and least expensive products
- Top-rated products
- Most reviewed products
- Relationship between discounts and reviews
- Relationship between ratings and reviews
- Comparison of high-discount and low-discount products
- Product category performance

# Dashboard Features
The Excel dashboard includes:
- Overview KPIs
- Product performance analysis
- Trend analysis
- Category analysis
- Interactive slicers for filtering data

# Key Findings

- Discount vs Reviews correlation: -0.137 - Higher discounts do not drive more reviews.
- Rating vs Reviews correlation: 0.057 — Higher-rated products tend to get more reviews.
- Most expensive product: 32PCS Portable Cordless Drill Set With Cyclic Battery Drive  at KSh 3,750.
- Least expensive product: 3PCS Single Head Knitting Crochet Sweater Needle Set at KSh 38.
- Products with High Discount avg rating: 3.61 vs Low Discount: 3.72 

# Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
